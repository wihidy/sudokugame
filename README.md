# sudokugame

A classic Sudoku game built with Python and Tkinter.  
Play at your own pace with multiple difficulty levels, a hint system, and a unique **visual solver** that shows the backtracking algorithm in action.

---

## ✨ Features

- **Three Difficulty Levels** – Easy (40 clues), Medium (30 clues), Hard (24 clues)
- **Backtracking Puzzle Generator** – Guarantees a single-solution puzzle every time
- **Visual Solver** – Watch the algorithm fill the board step by step with colour highlights
- **Move Counter** – Track how many numbers you’ve placed
- **Built‑in Timer** – See how long it takes to solve
- **Hint System** – Get one correct number revealed (with a visual cue)
- **Check Solution** – Verify your board and highlight correct/incorrect cells
- **Reset Board** – Start over from the original puzzle
- **Clean, intuitive GUI** with keyboard input (1‑9, Backspace/Delete)
